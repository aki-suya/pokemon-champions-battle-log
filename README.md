# Pokémon Champions Battle Log

ポケモン対戦の記録・統計・ダメージ計算を一体化したWebアプリです。

## 概要

対戦相手のパーティ、選出、持ち物、勝敗、メモを記録し、使用パーティごとの勝率や苦手な相手を確認できます。  
また、登録した自分のパーティ情報をもとに、技・特性・持ち物・天候・状態異常などを反映したダメージ計算も行えます。

## 主な機能

- 自分のパーティ登録
- 相手パーティ・選出・持ち物の記録
- 対戦履歴の保存・検索・編集・削除
- 使用パーティごとの勝率集計
- よく負ける相手・先発傾向の集計
- 技・特性・持ち物・天候・状態異常を反映したダメージ計算
- localStorageによるブラウザ保存

## 使用技術

- HTML
- CSS
- JavaScript
- localStorage

## 工夫した点

対戦記録とダメージ計算を別々のツールに分けるのではなく、1つの画面で連携できるようにしました。  
特に、登録した自分のパーティから攻撃側を選び、相手パーティから防御側を選んで計算できるようにした点を工夫しました。
